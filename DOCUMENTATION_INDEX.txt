🆘 TROUBLESHOOTING
================================================================================

If you encounter "Model not initialized":
   → Download model: python -c "from ultralytics import YOLO; YOLO('yolov8n.pt')"
   → Restart backend

If satellite API fails:
   → System continues with mock data
   → Add Copernicus credentials to .env to use real ESA API

If wave API times out:
   → Uses cached data if available
   → Automatic retry with exponential backoff

If database connection fails:
   → SQLite works automatically (no config needed)
   → For PostgreSQL, check connection string in .env

See PRODUCTION_DEPLOYMENT.md for more troubleshooting.
